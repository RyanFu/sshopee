# sshopee
## shopee数据助手

### 核心功能
* 后台在线产品汇总查询筛选(爬虫)
* ERP产品汇总查询筛选(总表 库存表)
* 刊登产品数据获取处理(赛盒API)
* 账号指标汇总统计(爬虫/统计)
* 异常订单汇总统计处理(爬虫)

### 辅助功能
* threading reuquests 多线程爬虫
* easyUI-edatagrid数据网格 Admin后台数据CRUD
* 表格导出上传更新数据
* 登录与权限管理

### 前端: 
* Jinja2 母版布局
* essyUI 页面细节UI
* javasript jquery 具体功能

### 后端:
* flask WEB框架
* waitress serve服务

### 数据库:
* sqlite3 数据存储
